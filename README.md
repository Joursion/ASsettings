### Android Studio 配置

###### KeyMap


| 快捷键             | 功能                           |
| --------------- | ---------------------------- |
| Alt + ;         | Toggle Presentation mode     |
| Ctrl + Alt + ;  | Toggle Distraction Free mode |
| Ctrl + ;        | Toggle Full Screen mode      |
| Ctrl + Alt + W  | Vim Emulator                 |
| Alt + Shift + . | Move Right                   |
| Alt + Shift + M | Move Down                    |
| Alt + Shift + , | Move To Opposite Group       |
